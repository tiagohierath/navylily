# Lesson thumbnails (optional)

Drop a 16:9 image here named after the lesson it belongs to, e.g.
`001.jpg` for `../001.md`. Supported extensions: jpg, jpeg, png, webp.

When present, `parser.sh` copies it into the served tree and shows it under the
lesson's first heading and as the lesson's thumbnail on the home page. Lessons
without an image here simply render as before — images are entirely optional.
