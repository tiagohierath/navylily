### Definição

As duas ideias que o autor considera as **mais importantes** de todo o aprendizado: ter **informação entrando** (input compreensível) e ter **feedback**. Sem as duas, você não fica bom de verdade.

### Input compreensível

O conceito vem do aprendizado de línguas. O autor demonstra ensinando palavras de uma língua artificial (Tokipona) mostrando o significado no contexto — você entende sem tradução. É assim que se aprende uma língua naturalmente.

Aplicado ao desenho: os **livros** são o input compreensível.

> "Os livros explicam de uma forma simplificada — é um passo antes da realidade. É como eu explicando a língua mostrando o significado."

É muito difícil aprender só observando a natureza crua; o livro é a versão "mastigada" que torna a informação compreensível (ver [[Usar Livros]] e [[Entrada de Informação Visual]]).

### Feedback

Feedback é simplesmente ver se você **acertou ou errou**.

> "Se você joga xadrez e nunca analisa as partidas, você não pega feedback — e nunca vai ficar bom de verdade."

No desenho, feedback é:

- **Analisar os próprios desenhos** ("como eu errei ao copiar essa foto?").
- **Comparar com outros artistas** ("como tal artista resolveria isso?").
- **Pedir conselho** a outras pessoas.

Isso fecha o loop do [[Método de Estudo]] (desenhar de memória → conferir erros → repetir).

### Adquirir x aprender (línguas)

O autor distingue **adquirir** (como um bebê pega a língua, automático) de **aprender** (estudar gramática conscientemente):

> "Adquirir é como entender uma piada sem ninguém te explicar; aprender é alguém te explicando a piada."

Ele atingiu 99% num teste de inglês só **assistindo YouTube na língua-alvo** por anos (pesquise os vídeos na própria língua). Estimativas dele: ~**10.000 palavras** para ler com fluência, ~**3.000** para falar; a curva é **exponencial** (quanto mais você sabe, mais rápido aprende). Para acelerar a aquisição, use **Anki / flashcards** — "uma das melhores técnicas de estudo de todos os tempos". O mesmo vale para desenho: imersão (aquisição) + estudo deliberado se somam.

### Conclusão

Informação entrando + feedback constante. Tenha as duas rodando e você tem 99% do que precisa.
